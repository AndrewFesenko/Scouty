## Building Documentation

Documentation is written in Markdown and can be viewed directly on GitHub or built into HTML:

```bash
# Install documentation tools (optional)
pip install mkdocs mkdocs-material

# Serve documentation locally
mkdocs serve

# Build static site
mkdocs build
```
