# Hardware - CAD Design

Placeholder for mechanical CAD files.

## Planned Components

### Robot Chassis
- Base platform
- Motor mounts
- Wheel attachments
- Battery compartment

### Sensor Mounts
- Camera mount (pan/tilt optional)
- Ultrasonic sensor brackets
- IMU mounting plate

### Electronics Enclosures
- Raspberry Pi case
- MCU enclosure
- Wire management

## Tools

Recommended CAD tools:
- Fusion 360
- SolidWorks
- FreeCAD (open source)
- OnShape

## Files

Place your CAD files here:
- Part files (.step, .stl)
- Assembly files
- Technical drawings (.pdf)
- 3D printing files (.stl, .gcode)

## Design Considerations

- Accessibility for maintenance
- Cable routing
- Cooling/ventilation
- Weight distribution
- Center of gravity
- Durability and impact resistance

## 3D Printing

If using 3D printing:
- Material: PLA, PETG, or ABS
- Infill: 20-30% for structural parts
- Layer height: 0.2mm recommended
- Support material where needed
